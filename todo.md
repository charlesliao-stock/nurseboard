# 優良護理人員看板產生器 - 開發清單

## 後端功能
- [x] 資料庫架構：看板記錄表（id, userId, theme, department, name, achievement, photoUrl, templateId, createdAt）
- [x] 資料庫架構：主題設定表（id, title, defaultValue）
- [x] LLM 文案生成 API：根據簡短描述產生 30 字內優良事蹟
- [x] 看板記錄 CRUD API：建立、讀取、更新、刪除看板記錄
- [x] Google Drive 上傳 API：上傳看板圖檔與照片至 Google Drive
- [x] 照片上傳 API：上傳照片至 S3 並取得 URL

## 前端功能
- [x] 雙欄式介面布局：左側控制區、右側預覽區
- [x] 版型選擇器：10 種粉色花卉風格版型快速切換
- [x] 資料輸入表單：主題、單位、姓名、優良事蹟（限 30 字）
- [x] LLM 文案生成按鈕：點擊自動生成優良事蹟文案
- [x] 照片上傳功能：點擊相機圖示上傳照片，即時預覽
- [x] 即時預覽區：根據輸入資料與選擇版型即時更新看板
- [x] 10 種版型設計：
  - [x] 版型 1：粉紅色 + 愛心圖示 + 圓形氣泡
  - [x] 版型 2：淡粉色 + 醫療十字 + 花卉裝飾
  - [x] 版型 3：珊瑚粉 + 愛心 + 浮動圓形
  - [x] 版型 4：玫瑰粉 + 十字 + 葉片裝飾
  - [x] 版型 5：蜜桃粉 + 混合圖示 + 波浪邊框
  - [x] 版型 6：淺粉紅 + 愛心 + 星星點綴
  - [x] 版型 7：紫粉色 + 十字 + 圓形浮動
  - [x] 版型 8：橙粉色 + 愛心 + 花朵裝飾
  - [x] 版型 9：莓粉色 + 十字 + 葉片浮動
  - [x] 版型 10：奶油粉 + 混合圖示 + 完整裝飾
- [x] 匯出功能：一鍵匯出高畫質橫向 A4 比例看板（PNG/JPG）
- [x] 上傳至 Google Drive：一鍵上傳看板圖檔與照片
- [x] 看板記錄列表：查閱已產生的看板記錄
- [x] 響應式設計：確保桌面端清晰可見
