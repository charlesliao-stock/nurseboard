## 快速開始

### 登入
1. 訪問應用首頁
2. 點擊「登入開始使用」按鈕
3. 使用 Manus OAuth 帳號登入

### 建立看板
1. 登入後，點擊「開始製作看板」或導航中的「看板產生器」
2. 填寫護理師資訊
3. 選擇喜愛的版型
4. 上傳照片
5. 匯出或上傳至 Google Drive

## 主要功能

### 1. 版型選擇
- **10 種粉色花卉風格版型**：每種版型擁有獨特的配色、圖示和裝飾元素
- **即時預覽**：選擇版型時，右側預覽區會立即更新

### 2. 資料輸入
- **主題**：預設為「2026優良護理人員」，可自訂
- **單位**：護理師所屬單位（如：手術室、加護病房等）
- **姓名**：護理師全名
- **優良事蹟**：限制 30 字，描述護理師的優良表現

### 3. AI 文案生成
- 點擊「AI 生成文案」按鈕
- 根據您輸入的簡短描述，系統會自動生成專業的 30 字內優良事蹟文案
- 生成的文案會自動填入優良事蹟欄位

### 4. 照片上傳
- 點擊預覽區中的相機圖示
- 選擇護理師照片（建議使用去背照或工作照）
- 照片會立即顯示在預覽區

### 5. 匯出看板
- **選擇格式**：PNG（推薦，無損）或 JPG（檔案較小）
- **點擊匯出按鈕**：看板將以高畫質（橫向 A4 比例）下載到您的電腦

### 6. 上傳至 Google Drive
- 點擊「上傳至 Google Drive」按鈕
- 系統會自動上傳看板圖檔和護理師照片
- 上傳完成後，您可在 Google Drive 中查看和分享

### 7. 看板記錄
- 點擊首頁右上角的「看板記錄」
- 查看所有已產生的看板
- 可下載或刪除歷史記錄

## 詳細步驟

### 步驟 1：填寫基本資訊
```
1. 主題：2026優良護理人員（可修改）
2. 單位：輸入護理師所屬單位
3. 姓名：輸入護理師全名
4. 優良事蹟：輸入 30 字以內的事蹟描述
```

### 步驟 2：使用 AI 生成文案（可選）
```
1. 在優良事蹟欄位輸入簡短描述
2. 點擊「AI 生成文案」按鈕
3. 系統會自動生成專業文案
4. 文案會自動填入欄位
```

### 步驟 3：上傳照片
```
1. 在預覽區找到相機圖示
2. 點擊相機圖示
3. 選擇護理師照片
4. 照片會立即顯示在預覽區
```

### 步驟 4：選擇版型
```
1. 在左側控制區找到版型選擇器
2. 點擊不同版型查看效果
3. 選擇最喜歡的版型
4. 預覽區會即時更新
```

### 步驟 5：匯出看板
```
1. 選擇匯出格式（PNG 或 JPG）
2. 點擊「匯出看板」按鈕
3. 看板會下載到您的電腦
4. 檔案名稱：nurse-board-[姓名]-[時間戳].png/jpg
```

### 步驟 6：上傳至 Google Drive
```
1. 點擊「上傳至 Google Drive」按鈕
2. 系統會上傳看板圖檔和照片
3. 等待上傳完成（可能需要 30 秒）
4. 上傳完成後會收到成功提示
5. 檔案會出現在您的 Google Drive 根目錄
```

## 常見問題

### Q1：優良事蹟超過 30 字會怎樣？
**A：** 系統會自動限制為 30 字。超出部分會被截斷，無法輸入。

### Q2：可以修改已保存的看板嗎？
**A：** 目前無法直接編輯已保存的看板。如需修改，請刪除舊記錄並建立新看板。

### Q3：照片上傳失敗怎麼辦？
**A：** 
- 檢查網路連接
- 確認照片檔案大小不超過 5MB
- 確認照片格式為 JPG 或 PNG
- 重新嘗試上傳

### Q4：匯出的看板解析度是多少？
**A：** 看板以 2 倍解析度匯出，確保高畫質。橫向 A4 比例（寬 1188px × 高 840px）。

### Q5：上傳至 Google Drive 需要授權嗎？
**A：** 是的。首次使用時，系統會要求您授權訪問 Google Drive。請按照提示完成授權。

### Q6：可以批量建立多個看板嗎？
**A：** 可以。每次填寫資訊並匯出即可建立新看板。系統會自動保存所有記錄。

### Q7：版型中的圖示可以自訂嗎？
**A：** 目前版型是預設的，無法自訂圖示。但可以選擇 10 種不同的版型組合。

### Q8：如何分享看板？
**A：** 
- 匯出看板後，可直接分享 PNG/JPG 檔案
- 上傳至 Google Drive 後，可在 Drive 中分享連結

### Q9：看板記錄會保存多久？
**A：** 看板記錄會永久保存在系統中，直到您主動刪除。

### Q10：支援哪些瀏覽器？
**A：** 支援 Chrome、Edge、Firefox、Safari 等現代瀏覽器。建議使用最新版本以獲得最佳體驗。
