# 💉 KMUH 班表輸入小幫手 (Chrome Extension)

支援跨月自動跳轉、多工作表選取與字典預檢的 KMUH 班表輸入輔助工具。

---

## 🔄 如何更新外掛 (Update Guide)

由於本外掛為自行開發（未上架 Chrome Web Store），更新時需要手動操作。請按照以下步驟進行：

### 1. 下載最新版本
當外掛提示有新版本時，點擊外掛視窗中的 **「📥 立即下載更新檔 (ZIP)」** 按鈕，下載最新版本的壓縮檔。

### 2. 解壓縮檔案
1. 找到下載的 `hrinput-main.zip`。
2. 將其解壓縮至目錄。
3. 將解壓縮後的目錄內的檔案，直接覆蓋掉您原本安裝外掛的資料夾。

### 3. 在 Chrome 中重新載入
1. 開啟 Chrome 瀏覽器，在網址列輸入 `chrome://extensions/` 並按下 Enter。
2. 找到並按下 **「更新」**的按鈕
3. 更新完成！您可以在外掛視窗標題確認版本號。

---

## ✨ 主要功能
- **跨月記憶**：自動記憶本月班表並跳轉至下個月。
- **Excel 匯入**：支援 `.xlsx` 格式，自動辨識職編與姓名。
- **班別預檢**：自動檢查 W+/N+ 等特殊班別，並提供更換建議。
- **字典管理**：自定義 Excel 班別與系統班別的對應關係。
- **在線修正**：線上修正班表後，可直接匯入班表。
